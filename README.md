# 여기여기 · 장소 다이어리

기획서(`project-plan-template.md`)를 바탕으로 만든 **개인 장소 북마크 다이어리** 웹앱(PWA)입니다.
내가 다녀온 장소를 사진·메모·별점·기분과 함께 기록하고, 지도/목록/달력으로 보고, 외부 지도 앱으로 바로 열 수 있습니다.

## 실행 방법

브라우저 보안 정책상 `file://`로 바로 열면 일부 기능(서비스워커 등)이 막히므로 **로컬 서버**로 실행하세요.

```bash
# 이 폴더에서
python -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

또는 Node가 있다면:

```bash
npx serve .
```

휴대폰에서 같은 와이파이로 접속한 뒤, 브라우저 메뉴의 **"홈 화면에 추가"** 를 누르면 앱처럼 설치됩니다(PWA).

## 구현된 기능 (기획서 대비)

| 기획 항목 | 상태 | 구현 내용 |
|---|---|---|
| 모바일 우선 PWA, 서버 0개 | ✅ | 정적 파일 + 서비스워커, 백엔드 없음 |
| 로컬 저장 + 백업 파일 | ✅ | IndexedDB 저장 / `.json` 백업 내보내기·불러오기 |
| 장소 추가 ①지도 선택 | ✅ | 에디터 지도를 눌러 핀 지정 (역지오코딩으로 주소 자동) |
| 장소 추가 ②검색 | ✅ | 장소명·주소 검색 (Nominatim) |
| 장소 추가 ③링크 붙여넣기 | ✅ | 구글/카카오/네이버 지도 링크에서 좌표 추출 |
| 기록: 100자 메모·사진·별점·기분·날짜 | ✅ | 메모 100자 제한, 사진 자동 압축 저장 |
| 달력(날짜별 보기) | ✅ | 월 달력 + 날짜 클릭 시 해당일 기록 |
| 이동 버튼(앱/웹 자동) | ✅ | 구글(앱·웹 자동)·카카오·네이버 / 앱 스킴 + 웹 폴백 |
| 카드 이미지 내보내기 | ✅ | 사진+메모+주소 카드 PNG (Canvas) |
| 제외: 로드뷰·길찾기·타인리뷰·공유서버 | ✅ | 미구현(범위 밖) |

## 기술 선택 메모

기획서는 카카오맵 API를 권장했지만, 그건 **API 키 발급이 필요한 블로커**입니다.
동일하게 무료·무서버이면서 **키가 필요 없는** 조합으로 대체했습니다.

- 지도 타일: **CARTO Voyager** (키 불필요) — 장소 이름·POI가 보이면서 색이 과하지 않은 중간 톤
- 장소 검색: **Photon (komoot)** (키 불필요) — **부분검색 지원** (예: "원광대" → 원광대학교)
- 저장: **IndexedDB** (사진은 압축 후 dataURL, 폴더는 별도 store)
- 카드 이미지: **HTML Canvas** / PDF: **jsPDF** / 폴더 저장: **File System Access API**(지원 브라우저)

> **솔직한 한계:** CARTO·Photon 모두 바탕 데이터는 OpenStreetMap입니다. *디자인*은 카카오급으로 깔끔하지만, 카카오맵 수준의 **국내 POI 밀도**는 키 없이는 원리적으로 불가합니다. 진짜 카카오급 데이터가 필요하면 **카카오맵 JS키**(무료 가입)나 **브이월드(VWorld) 키**를 발급해 베이스맵·검색을 교체하면 됩니다. 구조는 그대로 둬도 됩니다.

## 폴더(분류)

- 백업 탭의 **🗂️ 폴더 관리** 또는 기록 편집 화면의 **＋ 폴더 관리**에서 폴더를 추가/이름변경/색변경/삭제할 수 있습니다.
- 기본 6개(맛집·카페·여행·쇼핑·문화·자연)가 제공되며 자유롭게 바꿀 수 있습니다.
- 각 기록에 폴더를 지정하면 지도 마커 색, 카드 배지, 목록 상단 필터로 분류됩니다.
- 폴더를 삭제해도 그 폴더의 기록은 사라지지 않고 "폴더 없음"이 됩니다.

## 파일 구조

```
index.html              화면 마크업
css/styles.css          스타일 (모바일 우선)
js/db.js                IndexedDB 래퍼
js/app.js               앱 로직 (지도·검색·기록·달력·내보내기·백업)
manifest.webmanifest    PWA 매니페스트
sw.js                   서비스워커 (앱 셸 오프라인 캐시)
assets/icon-192.png     아이콘
assets/icon-512.png     아이콘
```

## 주의

- 모든 데이터는 **이 기기·이 브라우저에만** 저장됩니다(비공개). 브라우저 데이터를 지우면 사라지니, **백업 탭에서 가끔 `.json` 내보내기**를 해두세요.
- 지도 타일·장소 검색은 인터넷 연결이 필요합니다(앱 화면 자체는 오프라인에서도 열립니다).
- 검색 서비스(Nominatim)는 공용 무료 서버라 과도한 연속 검색은 제한될 수 있습니다.
